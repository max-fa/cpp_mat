# Welcome to CPPMat!

## Overview

The basic gist of this project is to numerically implement what I learned in undergraduate linear algebra. Things like matrix arithmetic, factorizations, row reduction, and eigenvalues/eigenvectors.

## Current State of the Project (3/2/2026)

At the moment, I've completed the full set of arithmetic operations (addition, subtraction, scaling, and matrix multiplication). I am currently working on row reduction.

## Longterm Plans

After completing the full set of matrix operations outlined above, I'm going to start optimizing my algorithms and the Matrix class for operational speed and memory efficiency.
I am also considering expanding the scope of the library beyond linear algebra and to start implementing the other math subjects I learned in my bachelor's (calculus, diff eq, vector calc, etc.).

## Documentation & API

At the moment, I'm working on a proper documentation system. Once I finish implementing row reduction, I'll write a more comprehensive set of docs. For now, just go to /tests to see the different operations
in use.
